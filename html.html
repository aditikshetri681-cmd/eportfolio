HTML
Structure

HyperText Markup Language defines the skeleton of every web page headings, paragraphs, forms, tables, links using a system of nested, semantic elements.

HTML5
<!-- Semantic HTML document structure -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8"/>
  <title>Aditi Kshetri — Portfolio</title>
</head>
<body>

  <header>
    <nav>
      <a href="#home">Home</a>
    </nav>
  </header>

  <main>
    <article>
      <h1>About Aditi</h1>
      <p>Junior Full Stack Developer</p>
      <ul>
        <li>HTML & CSS</li>
        <li>JavaScript</li>
        <li>Bootstrap</li>
      </ul>
    </article>
  </main>

  <footer>
    <p>&copy; 2025 Aditi Kshetri</p>
  </footer>

</body>
</html>
